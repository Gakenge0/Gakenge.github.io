C Programming Fundamentals

In Week Two, we were introduced to programming concepts using the C language, a foundational skill in embedded systems development.

Basic Concepts

  • Structure of a C program: headers, main function, and statements.
  • Variable declarations, data types, and scope.
  • Operators: arithmetic, relational, logical, and bitwise.
  • Control structures: if, else, switch.
  • Loops: for, while, and do-while.

Functions and Modular Code

  • Declaring and calling functions with parameters.
  • Understanding return values.
  • Importance of code reusability and modularization.

Hands-On Practice

We practiced writing simple programs such as calculators, conditional logic, and basic number manipulation. The focus was on logic, syntax, and debugging.
