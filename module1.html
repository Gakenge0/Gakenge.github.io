HTML and CSS

In Week One, we covered an introduction to the basic building blocks of a webpage, namely HTML and CSS.

HTML Basics

An HTML document starts with a declaration that informs the browser of the document type. The declaration <!DOCTYPE html> specifies that we are using HTML5.

Key topics covered include:

  • HTML structure and elements, defined by start and end tags.
  • Image insertions using the <img> tag.
  • Linking pages using the <a> tag.
  • Various attributes that provide additional information about elements.
  • Different heading levels from <h1> to <h6>.
  • Paragraphs and other text formatting elements.

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. Some fundamental CSS concepts include:

  • Selectors - used to target HTML elements (e.g., p { color: blue; }).
  • Properties and values - define how elements appear (e.g., background-color, font-size).
  • Inline, internal, and external stylesheets for applying CSS.
  • The box model - understanding margins, borders, padding, and content spacing.
  • Responsive design using media queries.
