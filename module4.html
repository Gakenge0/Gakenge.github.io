Interfacing with Sensors and Actuators

Week Four focused on enabling our embedded systems to interact with the physical world by connecting and controlling sensors and actuators.

What Are Sensors and Actuators?

  • Sensors detect physical phenomena such as temperature, light, sound, or motion.
  • Actuators convert electrical signals into physical actions (e.g., motors, buzzers, LEDs).

Practical Applications

  • Reading analog signals using sensors (e.g., temperature sensors, LDRs).
  • Using digital input from switches and pushbuttons.
  • Controlling LEDs, buzzers, and motors using output pins.
  • Understanding Pulse Width Modulation (PWM) for motor control and LED dimming.

Projects and Exercises

We built circuits to:

  • Light an LED when a sensor threshold is met.
  • Activate a buzzer using a button press.
  • Control servo motors using PWM signals.
