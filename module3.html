Microcontrollers and Tinkercad

In Week Three, we explored the basics of microcontrollers and used Tinkercad to simulate and build circuits virtually. This module laid the groundwork for hardware interaction using embedded systems.

Microcontroller Basics

  • Understanding what a microcontroller is and how it differs from a microprocessor.
  • Introduction to popular microcontrollers: Arduino, ATmega328, etc.
  • Pin configurations, I/O ports, and power supply.

Hands-On with Tinkercad

  • Setting up a virtual breadboard in Tinkercad Circuits.
  • Connecting components like LEDs, resistors, and switches.
  • Writing simple Arduino (C-based) code to control digital output pins.
  • Simulating button presses and LED blinking.

Learning Outcomes

By the end of the week, we could simulate a basic embedded system circuit, write simple control code, and understand the hardware-software relationship.
